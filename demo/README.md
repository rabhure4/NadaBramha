# Demo

Drop your demo videos and GIFs here.

Suggested files:

- `full-walkthrough.mp4` — Complete app walkthrough
- `voice-studio-demo.mp4` — Voice generation and download
- `podcast-lab-demo.mp4` — Timeline editing with background music
- `quick-preview.gif` — Short animated preview for the README

**To embed in GitHub README:**  
Upload the video to the repo, then use:
```md
https://github.com/<user>/<repo>/assets/<video-id>
```
Or drag-and-drop into a GitHub issue/PR to get an embeddable URL.
